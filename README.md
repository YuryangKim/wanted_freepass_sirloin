# 1조 Free Pass 설로인 프로젝트

## 👨‍💻 프로젝트 소개

원티드 X 프리온보딩 1주차 두번째 프로젝트로 설로인 상품등록 페이지 입니다.

<p>팀장: 이용우 <br>
팀구성: 김유량, 양주영, 이용우, 홍유진</p>

**배포주소** : <br>
**설치 및 시작방법**

- npm i 패키지 설치
- npm start 로컬서버 실행
